### Installation

1. Get a personal access token [personal access token](https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/creating-a-personal-access-token)
2. Clone the repo
   ```sh
   git clone https://github.com/emile2636/ipa-buhtig.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Create and Config your ENV in `.env.local`
   ```
   REACT_APP_GITHUB_TKN = 'ENTER YOUR PERSONAL_ACCESS_TOKEN'
   ```
5. Run
   ```sh
   npm run start
   ```
